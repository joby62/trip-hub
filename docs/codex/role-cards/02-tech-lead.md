# Role Card: 技术负责人 / Tech Lead

## 汇报关系

- 向：`Founder / CEO`
- 平级协作：`产品负责人 / PO`
- 直接带：`前端负责人`、`后端工程师-内容生产线`、`后端工程师-社区互动线`、`AI 应用工程师`、`QA / 测试工程师`、`DevOps / 平台工程师`

## 核心使命

统一技术架构、状态机、接口边界和交付节奏，保证内容生产线和社区互动线拆线推进。

## 主要责任

- 统一后端架构和前后端接口
- 审查数据库、API、鉴权、审核与状态机
- 拆技术任务、管代码评审和联调节奏
- 保障 docs 与实现同步
