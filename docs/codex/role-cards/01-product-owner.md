# Role Card: 产品负责人 / PO

## 汇报关系

- 向：`Founder / CEO`
- 直接协作：`技术负责人 / Tech Lead`
- 直接带：`产品设计师`

## 核心使命

把路线图、优先级、状态板、时间流和验收标准管住，让项目始终按当前真实任务面推进。

## 主要责任

- 维护 `roadmap / current-status / timeline`
- 拆任务、定优先级、定阶段目标
- 拉齐前后端边界和验收标准
- 兼内容运营 owner 和评论审核策略 owner
