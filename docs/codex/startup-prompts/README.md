# Startup Prompts

## 说明

这一组文件可以直接作为 Codex 新线程的启动提示使用。

## 推荐用法

1. 默认进入项目时，用：
   `default-project-startup.md`
2. 要进入具体岗位时：
   先看对应 `role-cards/*.md`
3. 然后把对应 `startup-prompts/*.md` 作为线程启动提示
