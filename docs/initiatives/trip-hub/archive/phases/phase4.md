# Phase 4

## 目标

先把站点骨架从“单页长卷”改成“4 主视图”，但不在这一阶段重做大部分视觉细节和内容编排。

这一阶段的任务是先把新的站点结构立起来，让后续所有功能和内容都有新的承载壳。

## 范围

- 建立 `总览 / 行程 / 景点 / 清单` 四个一级视图
- 重做顶部导航和底部导航
- 保留现有功能逻辑，不删任何模块
- 先把现有内容挂到新视图容器里

## Step By Step

1. 重组主页面容器
   把当前按 section 纵向堆叠的结构，改成四个一级视图容器。

2. 重做顶部导航
   顶部导航升级为新的 glass top bar，但先不做最终视觉收口。

3. 重做底部导航
   底部导航改为 `总览 / 行程 / 景点 / 清单` 四个一级入口。

4. 挂接现有模块
   把当前的总览、日程、工具、清单、图片、原文、搜索、避坑等模块迁移到对应一级视图里。

5. 保持旧功能继续可用
   搜索、避坑筛选、工具卡、详情层、灯箱、原文模式、深链不能断。

6. 做第一轮视图切换状态管理
   让四个一级视图之间的切换先成立，保证后续可以分视图逐步重构。

## 交付物

- 四个一级视图骨架
- 新的顶部 / 底部导航
- 旧功能仍可运行的站点壳

## 验收

- 页面已经不是单页长卷结构，而是 4 主视图结构。
- 当前功能没有消失，只是承载位置变了。
- 后续可以在新壳上继续逐视图深改。
