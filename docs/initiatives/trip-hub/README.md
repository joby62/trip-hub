# Trip Hub Initiative

这是当前仓库的主 initiative。

它覆盖 3 条核心产品线：

- 移动端旅行产品面
- `Word -> 豆包 -> JSON` 的内容生产后端线
- 评论、点赞、UP、踩的真实社区互动线

## Layout

- `product/`
  - 页面信息架构、移动端交互和体验方向
- `architecture/`
  - 当前运行结构、内容流水线、后端方案、地图链路、ADR
- `reviews/`
  - 预留给验收与专项评审
- `records/`
  - 预留给结项记录和里程碑记录
- `archive/`
  - superseded 或历史阶段资料
