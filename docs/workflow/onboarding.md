## 开始写代码前必须确认

1. 自己当前接的是哪条任务线
2. 这条任务线的 live initiative 文档是哪一份
3. 最近一次 currentness 更新在哪张面板里
4. 是否有相关 ADR
5. 改完后要同步哪些 docs

## 推荐节奏

### 第 1 小时

- 读 `docs/README -> initiatives/NOW -> initiatives/ROADMAP`

### 第 1 个半天

- 读岗位对应文档
- 本地把项目跑起来

- 明确自己所属任务线
- 明确主方案文档
- 明确回归范围
