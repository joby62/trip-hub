# Role Card: QA / 测试工程师

## 汇报关系

- 向：`技术负责人 / Tech Lead`
- 主要协作：`前端负责人`、`后端工程师-社区互动线`

## 核心使命

把移动端主路径、导入任务流、登录流、评论审核流和异常态回归制度化。

## 主要责任

- 维护测试用例和回归清单
- 验收接口和状态流
- 输出缺陷优先级和验收结论
