# Team Map

## 组织结构

```text
Founder / CEO
├── 产品负责人 / PO
│   └── 产品设计师
└── 技术负责人 / Tech Lead
    ├── 前端负责人
    │   └── 前端工程师
    ├── 后端工程师-内容生产线
    ├── 后端工程师-社区互动线
    ├── AI 应用工程师
    ├── QA / 测试工程师
    └── DevOps / 平台工程师
```

## 角色使用建议

- 要推进范围、优先级、验收：用 `产品负责人 / PO`
- 要推进架构、拆技术任务、做跨线评审：用 `技术负责人 / Tech Lead`
- 要推进移动端页面和联调：用 `前端负责人` 或 `前端工程师`
- 要推进 `Word -> 豆包 -> JSON`：用 `后端工程师-内容生产线`
- 要推进评论、点赞、UP、踩：用 `后端工程师-社区互动线`
- 要推进提示词、结构化输出、评测：用 `AI 应用工程师`
- 要推进测试和回归：用 `QA / 测试工程师`
- 要推进环境、数据库、worker、监控：用 `DevOps / 平台工程师`
- 要推进交互稿和视觉稿：用 `产品设计师`
