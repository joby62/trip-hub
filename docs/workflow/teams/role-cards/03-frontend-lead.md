# Role Card: 前端负责人

## 汇报关系

- 向：`技术负责人 / Tech Lead`
- 直接带：`前端工程师`
- 主要协作：`产品设计师`、`QA / 测试工程师`

## 核心使命

保证移动端 UI 架构、组件边界、状态接线和联调节奏稳定。

## 主要责任

- 管理移动端页面结构和组件规范
- 对接后端 API 和鉴权流程
- 保障景点流、详情页、评论流、上传流和异常态体验
