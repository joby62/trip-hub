# Role Card: DevOps / 平台工程师

## 汇报关系

- 向：`技术负责人 / Tech Lead`

## 核心使命

把 API、数据库、Redis、对象存储、worker 和监控告警搭成可部署、可观察、可恢复的环境。

## 主要责任

- 维护开发 / 测试 / 生产环境
- 维护 CI/CD、迁移、日志和监控
- 保障导入任务和社区写入链路稳定
