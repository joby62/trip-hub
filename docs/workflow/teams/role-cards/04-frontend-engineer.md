# Role Card: 前端工程师

## 汇报关系

- 向：`前端负责人`
- 协作：`产品设计师`、`QA / 测试工程师`

## 核心使命

把移动端页面、评论、上传、点赞、UP、踩和错误处理稳定落地。

## 主要责任

- 实现页面和交互
- 接评论 API 和登录态
- 处理异常态、审核态和回归问题
