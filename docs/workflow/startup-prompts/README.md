# Startup Prompts

这组文件是给新线程直接复制使用的启动提示，不是 initiative 真相文档。

## What This Folder Is For

- 给某个岗位开一个全新 Codex 线程
- 给 owner 开一个新的调度线程
- 给 Worker A / B / C 开新的执行线程
- 确保新线程会按正确顺序读 docs，而不是靠记忆乱进

- startup prompt 只负责“新线程如何进入上下文”
