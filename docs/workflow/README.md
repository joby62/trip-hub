# Workflow Docs

`docs/workflow/` 是这套仓库的“人和流程”系统。

## What Lives Here

- `governance/`
  - 文档状态系统
  - 协作决策 SOP
  - 分支治理
- `startup-prompts/`
  - 新角色线程的第一条启动提示
- `teams/`
  - 团队结构、产品线 prompt、owner / worker handoff、Tech Lead 直属下属协作规则
- `operations/`
  - 回归、运行和日常执行说明
- `templates/`
  - 文档模板

## What Does Not Live Here

- 产品真相
- architecture baseline
- backend design truth
- milestone record
- archive snapshot

这些都应该进 `docs/initiatives/`。
